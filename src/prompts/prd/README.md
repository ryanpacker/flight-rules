# PRD Prompts

Prompts to help create Product Requirements Documents.

## Available Prompts

| Prompt | Description |
|--------|-------------|
| `creation-conversational.md` | Interactive interview that walks through each PRD section |

## Usage

Copy the prompt into your AI assistant and follow the conversation. The AI will guide you through each section of the PRD template (`.flight-rules/docs/prd.md`).

## Output

The conversation produces a completed PRD that follows the standard template structure:

- Overview
- Goals
- Non-Goals
- User Stories
- Constraints
- Success Criteria
