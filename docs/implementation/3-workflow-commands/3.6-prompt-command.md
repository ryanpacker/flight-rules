# 3.6 Prompt Command

Command for iteratively refining prompts.

## Goals

- Help users improve prompt quality
- Support systematic prompt iteration
- Maintain prompt history for learning

## Commands

- `prompt.refine` — Iteratively improve a prompt

## Dependencies

- **Requires**: None
- **Enables**: Better prompts for any use case

---

## 3.6.1. prompt.refine Command

**Goal**: Iteratively refine and improve prompts

**Approach**:
- Accept initial prompt from user
- Analyze for clarity, specificity, and effectiveness
- Suggest improvements with rationale
- Allow iterative refinement cycles
- Track versions if desired

**Key Features**:
- Analyzes prompt structure and content
- Suggests specific improvements
- Explains reasoning behind suggestions
- Supports multiple refinement rounds

**Acceptance Criteria**:
- Provides actionable improvement suggestions
- Explains the "why" behind each suggestion
- Allows iterative refinement
- Final prompt is clearly presented

**Status**: ✅ Complete
