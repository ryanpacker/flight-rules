# 1.3 Impl Reconcile

Command for updating implementation documentation based on actual work done.

## Goals

- Update task statuses to reflect completed work
- Add missing tasks that were implemented but not documented
- Mark areas as complete when all tasks are done

## Constraints

- Must preserve existing task structure and numbering
- Should detect both additions and completions
- Needs to analyze code changes to understand what was implemented

## Dependencies

- **Requires**: Implementation overview must exist (`docs/implementation/overview.md`)
- **Enables**: `docs.reconcile` (uses this command)
